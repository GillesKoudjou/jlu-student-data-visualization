# jlu-student-data-visualization
Data analysis and visualization of student statistics from Justus Liebig University Giessen (JLU), focusing on Faculty 07 (Mathematics, Computer Science, Physics, Geography) over the last 10 years. The project explores trends in student enrollment using available university data and presents the results through statistical analysis/visualizations
